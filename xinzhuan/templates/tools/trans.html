Chinese Translation

The tool powered by Google translation engines for Chinese to English translation, send chinese text to api and get response, which includes post-translational text, synongms This translation tool includes online translator, translation dictionary, text-to-speech in a variety of languages, multilingual virtual keyboard, spell checker, Russian decoder, back translation, and email service.

URL

http://xinzhuan.info/api/tools/translation.json

Request parameters

    Sample request

    curl "http://xinzhuan.info/api/tools/translation.json" --data "text=中国报纸分析"

Response properties

  • code -
  • response -
    • result -
    • show_pinyin -

    • Sample response

    { "code": 0, "response": {"result": "China newspaper analysis", "pinyin": "Zhōngguó bàozhǐ fēnxī" } }
